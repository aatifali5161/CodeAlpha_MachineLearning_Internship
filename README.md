Tasks Overview

Task 1: Credit Scoring Model
Objective: Build a machine learning model to predict creditworthiness using customer financial data. Developed and compared Logistic Regression, Random Forest, and XGBoost models. XGBoost achieved the best performance with 92.4% accuracy and 0.89 ROC-AUC. Key predictive features included Annual Income, Credit History, and Loan Amount. Generated evaluation plots including confusion matrices, ROC curves, and feature importance charts.

Task 2: Emotion Recognition from Speech
Objective: Develop a deep learning model to classify human emotions from speech audio data. Used MFCC feature extraction with CNN/LSTM architecture to detect emotions like happy, sad, angry, and neutral. Evaluated model performance using accuracy, precision, recall, and confusion matrices on audio datasets.

Task 3: Handwritten Character Recognition
Objective: Build an OCR system using Convolutional Neural Networks to recognize handwritten characters and digits. Trained on MNIST/EMNIST dataset to classify alphanumeric characters with high accuracy. Implemented data augmentation and model optimization techniques to improve recognition performance.

Tech Stack: Python, Scikit-learn, TensorFlow/Keras, Pandas, NumPy, Matplotlib, Seaborn, Librosa, XGBoost, Jupyter Notebook

How to Run: Each task folder contains source code and results. Clone the repo, navigate to the task folder, and run the main Python file.
